# Jetson AI 智慧安全帽偵測系統

## 專題名稱
Jetson AI 智慧安全帽偵測系統

## 專題介紹
本專題使用 NVIDIA Jetson Orin Nano 串接攝影機，透過 AI 影像辨識技術，即時偵測人員是否正確配戴安全帽。若偵測到未配戴安全帽的人員，系統會發出警示並記錄違規資訊，可應用於工地、工廠、實驗室等需要安全管理的場域。

## 使用設備
- NVIDIA Jetson Orin Nano
- USB Camera 或 CSI Camera
- LED 警示燈
- 蜂鳴器

## 使用技術
- Python
- OpenCV
- YOLO
- Flask
- Git / GitHub
